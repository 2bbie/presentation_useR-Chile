\begin{frame}{De que me ha servido el uso de información espacial y raster en 'R'}
\begin{itemize}[<+->]
\item Dejar de utilizar software de pago como: ArcGis, Envi, Erdas, etc.
\item Procesar imágenes MODIS de todo Chile entre 2000-2012.
\item La superficie total de chile corresponde a 12 MM de pixeles para una resolución de 250m
\item Se utilizan 300 imagenes aprox, frecuencia de 16 dias
\item Se realiza un suavizado para cada pixel para la serie temporal ('lowess' o 'loess').
\item Se calculan los resumen estadistico (fivenum,boxplot.stats,etc) para la serie completa.
\item Se calcula diferentes indices vegetacionales y de sequía
\item ...
\end{itemize}
\end{frame}
